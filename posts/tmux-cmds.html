5 comandos de tmux que tienes que saber

título: 5 comandos de tmux que tienes que saber
descripción: Mejora tu productividad en la terminal con estos 5 comandos de tmux
fecha: Tue Jul 13 10:33:22 AM CDT 2021

  1. CTRL b + w

    Este comando te permite visualizar todas tus ventanas en todas las sesiones abiertas con tmux y cambiarte de una a otra, puedes utilizar las teclas de vim para desplazarte o las flechas y presionar enter para entrar en una ventana.

  2. CTRL b + s

    Si has hecho multiples sesiones de ssh en un sistema, probablemente has hecho CTRL b + d, seguido de algo como tmux attach -t otrasession. Ya no más, similar a CTRL b + w, este comando te permite ver las sesiones abiertas en tmux sin tener que desprenderte de una sessión y añadirte a otra manualmente.

  3. CTRL b + [

    Con este comando entrarás en modo de selección, puedes desplazarte con teclas de vim (si las tienes activadas) o con las flechas, para copiar apreta la barra espaciadora y colorea el texto necesario apreta enter para mandarlo al buffer.

  4. CTRL b + ]

    Este comando te permite pegar el contenido del buffer de tmux, si copiaste desde vim recuerda que ese buffer esta dentro de esa aplicación unicamente.

  5. CTRL b + CTRL o

    Si tienes dos o más páneles dentro de una ventana y deseas rotarlos y cambiarte al siguiente panel al mismo tiempo, utiliza esta combinación, el caso de uso principal es cuando se tiene un editor de texto como vim en un pánel y una terminal en otro y necesitas cambiarte entre estos dos constantemente.
